# Bank Management System (Java)

A console-based Bank Management System developed in Java using Object-Oriented Programming concepts.

## Features

- Create New Account
- Deposit Money
- Withdraw Money
- Display All Accounts
- Delete Account
- Menu Driven Interface

## Technologies Used

- Java
- OOP
- ArrayList
- Scanner Class

## Concepts Used

- Classes & Objects
- Constructors
- Encapsulation
- Methods
- Loops
- Switch Case
- Collections (ArrayList)

## How to Run

1. Open the project in IntelliJ IDEA, Eclipse, or VS Code.
2. Compile all Java files.
3. Run `Main.java`.
